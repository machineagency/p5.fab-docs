    Resources #

    Here you’ll find some resources for using p5.fab, including guides which introduce key concepts, tutorials which put these concepts into practice, and examples to motivate more uses!

    The resources here are growing; if there’s something you’d like to see added or contribute, reach out on github!

    Guides #

    A Gentle Introduction to Digital Fabrication #

    A (very!) brief overview of some key digital fabrication concepts, including: G-Code, coordinate systems, machines, and their firmware. Intended for anyone new to using machines like 3D printers.

    A Gentle Introduction to Creative Coding with p5.js #

    A (very!) brief overview of some key creative coding concepts, including: how to sketch with p5.js. Intended for anyone new to creative coding.

    An overview of the p5.fab library!
