<h2 class="ref-header">setAbsolutePositionXYZ()</h2>
Set X/Y/Z axes to absolute position, relative to the origin of the machine, without changing the extruder positioning mode.

### Syntax
```
fab.setAbsolutePositionXYZ()
```

### Parameters
None
