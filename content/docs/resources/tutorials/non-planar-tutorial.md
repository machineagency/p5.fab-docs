## Non-Planar Printing
Check out the non-planar example in the <a href="https://bsubbaraman.github.io/test-interface/" target="_blank"> online editor</a>. It shows how to move out of the x-y plane while printing. If you're comfortable with this, see the <a href="../interactive-cube-tutorial">interactive cube</a> tutorial to see how to use a MIDI controller to change printing parameters.
